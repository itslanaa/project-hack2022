Dark Version

To have dark mode enabled, Keep your body element with data attribute data-layout-mode="dark" data-layout-mode="dark" data-topbar="dark" in the views/layout.ejs
E.g. <body data-layout-mode="dark" data-layout-mode="dark" data-topbar="dark"> to have Dark layout.
