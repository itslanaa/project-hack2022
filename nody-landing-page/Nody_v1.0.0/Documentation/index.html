Introduction

Nody is a multi purpose Bootstrap 4 landing page template.This responsive template comes with 5 Demo variations which can be used to build your product, service, marketing, app and SaaS landing pages. It’s fully responsive and built with bootstrap v4.4.1. You will be easily able to customize it for your needs.

Please feel free to get back to me in case if you are having any question or feedback.
